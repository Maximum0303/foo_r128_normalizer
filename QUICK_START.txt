R128 リアルタイム音量ノーマライザー v1.4.0
かんたん導入ガイド

一般利用者向け:

1. GitHub Releasesから次のファイルをダウンロードします。
   foo_r128_normalizer_v1.4.0.fb2k-component

2. ダウンロードしたファイルを開きます。

3. foobar2000の確認画面に従ってインストールします。

4. foobar2000を再起動します。

5. DSP設定を開き、
   「R128 音量ノーマライザー」を使用中のDSPへ追加します。

6. 最初は「自然 -18」を選びます。

一般利用では、Visual Studioやfoobar2000 SDKは必要ありません。

迷った場合:
・自然に音量を整える → 自然 -18
・少し力強くする → パワー -14
・静かで余裕を持たせる → リラックス -23
・夜間の急な大音量を抑える → ナイト -22

モダン -9、自動1帯 -10、自動3帯 -10は、
音色やダイナミクスにも作用する追加処理です。

比較方法:
・「音量一致」をオン
・「比較（押している間）」を押し続ける
・音量差に惑わされにくい状態で処理前後を比較できます

診断:
・「診断」タブで現在の測定値を確認
・「診断コピー」で結果をクリップボードへコピー
・用語が分からない場合は「用語集」を開く

確認済み表示環境:
・1920 x 1080
・Windows表示倍率150%

別の解像度や表示倍率では見え方が異なる場合があります。
