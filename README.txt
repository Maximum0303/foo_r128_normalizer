R128 リアルタイム音量ノーマライザー v1.5.2

通常の利用にVisual Studioやfoobar2000 SDKは必要ありません。

インストール:
1. foo_r128_normalizer_v1.5.2.fb2k-componentを開く
2. foobar2000へインストール
3. foobar2000を再起動
4. DSP Managerで「R128 音量ノーマライザー」を追加
5. 最初は「自然 -18」を選択

設定画面を直接開く:
Playback → R128 音量ノーマライザーの設定...

DSPへ追加した後は、DSP Managerを経由せず設定画面を開けます。
このメニューコマンドはキーボードショートカットにも割り当て可能です。
直接開いた画面はfoobar2000より手前を保ち、背後のfoobar2000も操作できます。
ほかのアプリに対する常時最前面にはなりません。

本DSPが未登録または複数登録されている場合は、
安全のため設定を変更せず案内を表示します。

画面レイアウト:
・基本設定と診断は、左列を上から下へ読んでから右列へ移る構成
・追加処理は、モダン／1バンド／3バンドの横3列カード表示
・共通の処理強度設定は追加処理タブの下段へ配置
・診断ページは上下の余白が均等になるよう調整

診断表示:
・2列レイアウトで各項目を正式名称表示
・Integratedラウドネス、True Peak、各減衰量も省略せず表示

表示テーマ:
・foobar2000本体のライト／ダーク設定へ自動追従
・設定画面、タブ、入力欄、ボタン、ツールチップ、用語集、
  項目ヘルプ、ライセンス・クレジット表示に対応

標準ノーマライズ:
・自然 -18
・パワー -14
・リラックス -23
・ナイト -22

追加マスタリング処理:
・モダン -9
・1バンド・アダプティブ -10
・3バンド・アダプティブ -10

追加マスタリング処理は、音色やダイナミクスにも影響します。

確認済み表示環境:
・1920 x 1080
・Windows表示倍率150%

動作対象:
・foobar2000 2.x
・Windows x64
