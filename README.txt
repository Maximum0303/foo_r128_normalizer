R128 リアルタイム音量ノーマライザー v1.4.0
安定公開版

このファイルは、完成済みのfoobar2000コンポーネントです。
通常の利用にVisual Studioやfoobar2000 SDKは必要ありません。

インストール:
1. foo_r128_normalizer_v1.4.0.fb2k-componentを開く
2. foobar2000の確認画面に従ってインストール
3. foobar2000を再起動
4. DSP設定で「R128 音量ノーマライザー」を追加
5. 最初は「自然 -18」を選ぶ

推奨する最初の設定:
自然 -18

標準ノーマライズ:
・自然 -18
・パワー -14
・リラックス -23
・ナイト -22

追加マスタリング処理:
・モダン -9
・自動1帯 -10
・自動3帯 -10

追加マスタリング処理は、音色やダイナミクスにも影響します。
音量統一を優先する場合は、標準4プリセットを使用してください。

確認済み表示環境:
・1920 x 1080
・Windows表示倍率150%
・設定画面全体が作業領域内に収まることを実機確認

その他の解像度や表示倍率では、見え方が異なる場合があります。

動作対象:
・foobar2000 2.x
・Windows x64

主な機能:
・R128ベースのリアルタイム音量補正
・Momentary／Short-term／Integrated／LRA
・True Peak推定と先読みリミッター
・曲頭安定化、静音保護、ゲイン固定
・モダン、1バンド、3バンドの追加処理
・音量一致A/B比較
・停止後の診断結果保持
・診断コピー、用語集、項目別ヘルプ
・異常値保護
・3タブ設定画面

既知の制限:
・3バンド処理は軽量・低遅延を優先した簡易構成
・業務用マスタリングの代替ではない
・True Peakは4倍オーバーサンプリングによる近似
・すべての解像度とDPI倍率を保証するものではない
・Windows x64専用
