# v1.4.0 — 最初の安定公開版

R128ベースのリアルタイム音量ノーマライザーです。

## ダウンロード

一般利用者は、次のファイルだけをダウンロードしてください。

`foo_r128_normalizer_v1.4.0.fb2k-component`

Visual Studioやfoobar2000 SDKは不要です。

## インストール

1. `.fb2k-component`ファイルを開く
2. foobar2000へインストール
3. foobar2000を再起動
4. DSP設定で「R128 音量ノーマライザー」を追加
5. 最初は「自然 -18」を選択

## 主な内容

- R128ベースのリアルタイム音量補正
- 7種類のプリセット
- True Peak推定と先読みリミッター
- 音量一致A/B比較
- 1バンド／3バンド・アダプティブ処理
- 診断表示、用語集、項目別ヘルプ
- 1920 × 1080、表示倍率150%で設定画面を実機確認

追加処理プリセットは、音色やダイナミクスにも影響します。
純粋な音量統一を優先する場合は標準4プリセットを使用してください。
