# v1.5.0 — 設定画面の直接起動

## ダウンロード

一般利用者は次をダウンロードしてください。

`foo_r128_normalizer_v1.5.0.fb2k-component`

Visual Studioやfoobar2000 SDKは必要ありません。

## v1.5.0の変更

- foobar2000本体のライト／ダーク表示へ自動追従
- 設定画面、各コントロール、タブ、ツールチップ、用語集をダークモード対応
- `Playback → R128 音量ノーマライザーの設定...`を追加
- DSP Managerを経由せず、現在有効な設定画面を直接表示
- メニューコマンドをキーボードショートカットへ割り当て可能
- 直接開いた設定画面をfoobar2000所有のモードレス画面へ変更
- 設定画面をfoobar2000より手前に保ったまま、foobar2000を操作可能
- Windows全体の常時最前面にはしない
- 二重起動を防止し、再実行時は既存の設定画面を前面表示
- DSP未登録時は追加方法を案内
- 同じDSPが複数登録されている場合は誤変更を防止
- 設定画面を開いている間にDSPチェーンが変わった場合も誤適用を防止
- 音声処理、プリセット値、GUID、設定形式v7は変更なし

## インストール

1. `.fb2k-component`ファイルを開く
2. foobar2000へインストール
3. foobar2000を再起動
4. DSP Managerで「R128 音量ノーマライザー」を追加
5. 以後はPlaybackメニューから直接設定

## 動作環境

- foobar2000 2.x
- Windows x64
