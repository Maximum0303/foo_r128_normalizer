# v1.5.1 — ヘルプ／ライセンス画面のテーマ追従

## ダウンロード

一般利用者は次をダウンロードしてください。

`foo_r128_normalizer_v1.5.1.fb2k-component`

Visual Studioやfoobar2000 SDKは必要ありません。

## v1.5.1の変更

- 項目ヘルプを専用ダイアログへ変更
- ライセンス・クレジット表示を専用ダイアログへ変更
- 両画面がfoobar2000本体のライト／ダーク／レトロ表示へ自動追従
- 長文をスクロール可能
- ヘルプ／ライセンス画面をコンパクト化
- 表示テキストを選択・コピー可能
- 音声処理、プリセット値、GUID、設定形式v7は変更なし

## インストール

1. `.fb2k-component`ファイルを開く
2. foobar2000へインストール
3. foobar2000を再起動
4. DSP Managerで「R128 音量ノーマライザー」を追加
5. Playbackメニューから設定

## 動作環境

- foobar2000 2.x
- Windows x64
