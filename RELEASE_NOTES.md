# v1.5.3 — プリセット名称の統一

## 変更内容

- プリセットボタンの表示名をバルーンの正式名称へ統一
- 選択中プリセット表示と文書も同じ名称へ統一
- ［初期設定］を押した際に確認ダイアログを表示
- 確認ダイアログはfoobar2000の表示テーマへ自動追従
- 確認文に、［適用］で再生中のDSPへ反映されることを明記
- キャンセル時は設定と［適用］状態を変更しない
- ［適用］はプリセット選択・数値変更・設定チェック変更時だけ有効
- 適用成功後は［適用］を無効へ戻し、連続適用を防止
- 適用済みのまま［OK］を押した場合は二重適用せず終了
- ナチュラル -18
- パワーブースト -14
- リラックス -23
- ナイトセーフ -22
- モダンブースト -9
- 1バンド・アダプティブ -10
- 3バンド・アダプティブ -10
- プリセット値、音声処理、GUID、設定形式v7は変更なし

## インストール

1. `foo_r128_normalizer_v1.5.3.fb2k-component`を開く
2. foobar2000へインストール
3. foobar2000を再起動

## 動作環境

- foobar2000 2.x
- Windows x64
