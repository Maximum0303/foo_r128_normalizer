# v1.5.4 — リアルタイム自動制御

## 変更内容

- 処理負荷をリアルタイムで監視する自動制御を全プリセットへ追加
- 要調整相当の状態が0.75秒続いた場合に自動安全補正を開始
- 自動安全補正ゲインを最大6 dBまで追加
- 安全な状態が3秒続くまで補正を保持
- 安全確認後は4秒の時定数で補正をゆっくり解除
- 「処理総合評価」を「現在の処理状態」へ変更
- 現在状態を「正常／要調整／自動調整中／調整上限」で表示
- 自動制御によって安全範囲へ戻った場合は「正常」へ自動復帰
- トラック中の過去の最大処理量は各最大値と0 dBTP超過イベントへ記録
- 標準プリセットにも自動安全補正を適用

## 互換性

設定形式v7、GUID、プリセット設定値、DLL名は変更していません。

## 動作環境

- foobar2000 2.x
- Windows x64
