# v1.5.2 — GUI表記・レイアウト改善

## ダウンロード

一般利用者は次をダウンロードしてください。

`foo_r128_normalizer_v1.5.2.fb2k-component`

Visual Studioやfoobar2000 SDKは必要ありません。

## v1.5.2の変更

- 追加処理タブをモダン／1バンド／3バンドの横3列カード表示へ変更
- 共通の処理強度設定を下段の横一杯へ配置
- 診断ページの列見出しと項目位置を調整し、上下の余白を均等化
- 基本設定と診断を、左列を上から下へ読んでから右列へ移る配置に変更
- 各列に役割見出しを追加し、視線移動を明確化
- 用語集の用語名も、できる範囲で正式名称へ統一
- 用語集の左側一覧幅を広げ、説明欄とのバランスを再調整
- 基本設定の数値項目とチェック項目を正式名称で表示
- 基本設定のラベル幅を拡張し、入力欄と単位を再配置
- 基本設定の項目ヘルプ見出しも正式名称へ統一
- 診断ページを3列から2列へ再配置
- 診断項目の略称を正式名称へ変更
- 入力／出力Integratedラウドネス、処理後True Peak、各減衰量などを省略せず表示
- 3バンド・アダプティブ減衰量も2列内に正式名称で表示
- 項目ヘルプの見出しも画面上の正式名称へ統一
- 音声処理、測定計算、プリセット値、GUID、設定形式v7は変更なし

## インストール

1. `.fb2k-component`ファイルを開く
2. foobar2000へインストール
3. foobar2000を再起動
4. DSP Managerで「R128 音量ノーマライザー」を追加
5. Playbackメニューから設定

## 動作環境

- foobar2000 2.x
- Windows x64
